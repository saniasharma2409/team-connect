# TeamConnect

A modern **Employee Directory & Dashboard** built using **HTML, CSS, and Vanilla JavaScript**.

**TeamConnect** helps organizations introduce new employees to the team by providing a centralized employee directory where everyone can discover colleagues, search employee profiles, and view important employee information through a clean and interactive dashboard.

The project demonstrates practical frontend development skills including authentication, DOM manipulation, LocalStorage, search and filtering, dynamic rendering, theme switching, and multi-page application development.


# Project Preview

> A responsive employee dashboard that allows users to register, create employee profiles, browse team members, and personalize their experience with Light/Dark mode.

# Why TeamConnect?

New employees often find it difficult to get familiar with their colleagues during onboarding.

**TeamConnect** solves this problem by providing a centralized employee workspace where team members can introduce themselves, explore employee profiles, search for colleagues, and access employee information through an interactive dashboard.

The project focuses on creating a simple, intuitive, and visually appealing employee directory while demonstrating core JavaScript development concepts.
